# Memoria Episódica

Esta carpeta almacena eventos y contextos específicos.

## Propósito
- Registrar experiencias detalladas
- Capturar el contexto completo de interacciones
- Facilitar el aprendizaje basado en experiencias

## Características
- Almacena eventos con detalles contextuales
- Permite reconstruir secuencias de interacción
- Útil para análisis retrospectivo y mejora continua
